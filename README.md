# BIT4139 – Week 6: Vulnerability Assessment and Management

## Overview

This repository contains the practical work completed for Week 6 - Vulnerability assessment and management. The practicals focused on understanding the vulnerability management lifecycle, identifying and assessing security weaknesses, developing vulnerability assessment plans, and proposing appropriate mitigation strategies. The activities also emphasized the importance of continuous monitoring, patch management, and effective reporting in maintaining secure information systems.

The repository demonstrates both the theoretical concepts and practical exercises carried out during the week's laboratory sessions and serves as a record of the skills and knowledge gained in vulnerability assessment and management.

The uploaded documentation includes:

- Screenshots

- Recommendations


## Topics Covered

- Vulnerability Management Framework
- Organizational Vulnerability Assessment Planning
- Patch Management
- Continuous Monitoring
- Vulnerability Tracking and Reporting
